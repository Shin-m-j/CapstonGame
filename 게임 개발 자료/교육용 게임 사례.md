# 사이버 보안 교육용 게임 사례 조사 보고서

## 1. 조사 목적 및 의의

본 조사는 현재 국내외에서 활용되고 있는 다양한 사이버 보안 교육용 게임 사례를 수집하고 분석하는 데 목적이 있다. 정보보호 지식을 단순 전달하는 방식을 넘어, 게임화(Gamification) 요소를 통해 학습자의 몰입도를 높이고 실질적인 보안 의식을 고취하는 최신 트렌드를 파악하고자 한다.

---

## 2. 주요 사이버 보안 교육 게임 사례 분석

### 2.1. 웹 기반 인터랙티브 퀴즈형

- **사례:** **Google Jigsaw Phishing Quiz**
- **기술 기반:** **Web Standard (HTML5 / React / JavaScript)**
- **특징:** 별도의 게임 엔진보다는 고도의 웹 인터페이스(UI)를 구현하여 실제 이메일 서비스와 동일한 경험을 제공하는 데 집중함.
- **링크:** [https://phishingquiz.withgoogle.com/](https://phishingquiz.withgoogle.com/)

### 2.2. 아케이드 및 미니게임 모음형

- **사례:** **Google Interland (Be Internet Awesome)**
- **기술 기반:** **Web-based 3D (WebGL / Three.js / PlayCanvas)**
- **특징:** 브라우저에서 실행되는 고퀄리티 3D 그래픽 게임으로, 저사양 기기에서도 접근 가능하도록 웹 최적화가 잘 되어 있음.
- **링크:** [https://beinternetawesome.withgoogle.com/ko_kr/interland](https://beinternetawesome.withgoogle.com/ko_kr/interland)

### 2.3. 해킹 및 방어 시뮬레이션형

- **사례:** **Hacknet**
- **기술 기반:** **C# / MonoGame (XNA 프레임워크 계열)**
- **특징:** 설치형 PC 게임으로, 실제 리눅스 터미널 환경과 유사한 인터페이스를 구현하기 위해 전용 게임 엔진 프레임워크를 사용함.
- **링크:** [Hacknet 스팀 페이지](https://store.steampowered.com/app/365450/Hacknet/)

### 2.4. 시나리오 기반 스토리텔링형

- **사례:** **PBS NOVA: Cybersecurity Lab**
- **기술 기반:** **Web-based (HTML5 Canvas / CreateJS)**
- **특징:** 애니메이션과 인터랙티브 요소를 효과적으로 전달하기 위해 2D 캔버스 라이브러리를 활용한 웹 게임.
- **링크:** [https://www.pbs.org/wgbh/nova/labs/lab/cyber/](https://www.pbs.org/wgbh/nova/labs/lab/cyber/)

### 2.5. 전문 보안 교육 플랫폼형

- **사례:** **CyberStart**
- **기술 기반:** **Web-based SaaS (Vue.js / Node.js 기반 시뮬레이션)**
- **특징:** 게임 엔진보다는 대규모 교육 데이터를 관리하고 가상의 서버 환경을 시뮬레이션하는 웹 플랫폼 형태.
- **링크:** [https://cyberstart.com/](https://cyberstart.com/)

---

## 3. 기술적 특징 및 트렌드

1. **웹(Web-based) 중심의 개발:** 교육용 콘텐츠 특성상 설치의 번거로움을 피하기 위해 **HTML5, WebGL**을 활용한 브라우저 기반 게임이 주류를 이룸.
2. **3D 라이브러리 활용:** 과거 플래시(Flash) 기반 게임들이 사라지고, **Three.js**나 **Babylon.js** 같은 자바스크립트 기반 3D 엔진으로 대체됨.
3. **전문가용은 전용 엔진 사용:** 기술적 깊이가 필요한 시뮬레이션(예: Hacknet)은 시스템 자원 활용을 위해 **C#, C++** 기반의 전용 게임 엔진(Unity, MonoGame 등)을 선호함.

---

## 4. 결론 및 시사점

조사 결과, 교육용 보안 게임은 **'접근성(Web)'**과 **'몰입감(Engine-based)'** 사이에서 타겟 층에 따라 기술적 선택을 달리하고 있다. 대중 교육용은 웹 표준 기술을 적극 활용하여 확산성을 높이고 있으며, 전문 교육용은 실제 환경과 유사한 시뮬레이션 기능을 강화하는 방향으로 발전하고 있다.
