# Capston_Design_GameProject

## 1. 과제 개요

### 1.1 과제 선정 배경 및 수행 목적

최근 보이스피싱, 스미싱, 사회공학적 기법 등 일상을 파고드는 사이버 위협은 더욱 교묘하게 진화하고 있다. 국민 대다수가 보안 위기감을 느끼고 있음에도 불구하고, 실제 조사 결과에 따르면 이러한 정교한 공격에 대응할 수 있는 실무 지식은 매우 부족한 실정이다. 기존의 교육 자료나 영상은 대개 딱딱한 형식을 띠고 있어 학습자의 흥미를 유발하기 어렵고, 이는 실제 범죄 상황에서의 대응력 부재로 이어진다.

본 과제는 이러한 한계를 극복하기 위해 사용자가 직접 '판별자'가 되어 위협 시나리오를 체험하고 대응법을 익히는 교육용 게임 개발을 목적으로 한다. "That's Not My Neighbor"의 추리 및 판별 메커니즘을 차용하여, 일상적인 연락망 속에서 숨어든 피싱 메일, 의심스러운 링크, 보이스피싱 수법을 가려내는 경험을 제공한다. 이를 통해 사용자가 위기 상황에서 올바른 대처법을 직관적으로 습득하고, 실생활에서의 보안 의식을 내재화하도록 돕고자 한다.
