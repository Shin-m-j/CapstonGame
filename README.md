# Project_B

## 과제 선정 배경 및 과제 수행 목적

최근 사이버 보안 위협은 지능형 지속 위협(APT), 사회공학적 기법(Social Engineering), 랜섬웨어 등 다양한 형태로 진화하며 점점 더 정교해지고 있다. 그러나 일반 사용자들에게 보안 교육은 여전히 ‘어렵고 딱딱한 것’으로 인식되는 경우가 많아, 실제 위협 상황에 직면했을 때 적절히 대응하지 못하는 ‘보안 불감증’ 문제가 심화되고 있다. 특히 기존의 보안 교육은 텍스트나 영상 중심의 일방향적 방식이 대부분이어서 학습자의 흥미와 참여를 유도하기 어렵고, 사이버 공격의 위험성을 현실적으로 체감하기에도 한계가 있다.

한편 피싱, 스미싱 등 일상에서 발생하는 사이버 범죄가 급증함에 따라 비전문가도 쉽게 이해할 수 있는 실용적인 보안 교육 방식의 필요성이 커지고 있다. 단순히 보안 수칙을 암기하는 방식만으로는 실제 상황에서의 대응 능력을 배양하기 어렵기 때문에, 보다 능동적이고 체험적인 학습 방식이 요구된다. 이러한 측면에서 게임의 요소와 메커니즘을 교육에 접목하는 방식은 효과적인 대안이 될 수 있다. 게임 기반의 학습은 사용자의 참여와 몰입을 극대화하여 복잡한 개념을 자연스럽게 이해하도록 돕고, 가상 환경 속에서 다양한 상황을 경험하게 함으로써 보안 위협에 대한 경각심을 제고할 수 있기 때문이다.

본 과제는 이러한 문제를 해결하기 위해 해킹 기법과 대응 방식을 결합한 교육용 게임 콘텐츠 개발을 목표로 한다. 게임 내에서는 바이러스의 감염 및 확산 경로, 백신의 구동 원리 등 추상적인 보안 개념을 직관적인 그래픽과 상호작용을 통해 시각적으로 구현하여 학습자의 이해를 돕는다. 또한 사용자가 방어자의 역할을 수행하는 시나리오를 구성하여, 사소한 보안 취약점이 초래할 수 있는 실제적인 결과를 간접적으로 체험하도록 설계한다.

나아가 실제 해킹 사례를 기반으로 한 시나리오를 통해 일상생활에서 실천 가능한 보안 수칙을 자연스럽게 체득하게 하고자 한다. 본 프로젝트는 캡스톤 디자인의 취지에 맞게 네트워크 통신 개념과 보안 알고리즘을 게임 로직에 적용함으로써, 학술적 공학 지식을 실무적인 교육용 소프트웨어로 구현하는 데 최종 목적을 둔다.
